<center><a href="/wiki/Isaac_Newton" title="Isaac Newton">Isaac Newton</a></center>
<p><a href="/wiki/File:GodfreyKneller-IsaacNewton-1689.jpg" class="image" title="Isaac Newton"><img alt="Isaac Newton" src="//upload.wikimedia.org/wikipedia/commons/thumb/3/39/GodfreyKneller-IsaacNewton-1689.jpg/200px-GodfreyKneller-IsaacNewton-1689.jpg" width="200" height="275" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/3/39/GodfreyKneller-IsaacNewton-1689.jpg/300px-GodfreyKneller-IsaacNewton-1689.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/3/39/GodfreyKneller-IsaacNewton-1689.jpg/400px-GodfreyKneller-IsaacNewton-1689.jpg 2x" data-file-width="407" data-file-height="559" /></a><br /></p>

<p><b><a href="/wiki/Mathematics" title="Mathematics">Mathematics</a></b> is the study of <a href="/wiki/Number" title="Number">numbers</a>, <a href="/wiki/Quantity" title="Quantity">quantity</a>, <a href="/wiki/Space" title="Space">space</a>, <a href="/wiki/Structure" title="Structure">structure</a>, and <a href="/wiki/Calculus" title="Calculus">change</a>. Mathematics is used throughout the world as an essential tool in many fields, including <a href="/wiki/Natural_science" title="Natural science">natural science</a>, <a href="/wiki/Engineering" title="Engineering">engineering</a>, <a href="/wiki/Medicine" title="Medicine">medicine</a>, and the <a href="/wiki/Social_sciences" title="Social sciences" class="mw-redirect">social sciences</a>. <a href="/wiki/Applied_mathematics" title="Applied mathematics">Applied mathematics</a>, the branch of mathematics concerned with application of mathematical knowledge to other fields, inspires and makes use of new mathematical discoveries and sometimes leads to the development of entirely new mathematical disciplines, such as <a href="/wiki/Statistics" title="Statistics">statistics</a> and <a href="/wiki/Game_theory" title="Game theory">game theory</a>. Mathematicians also engage in <a href="/wiki/Pure_mathematics" title="Pure mathematics">pure mathematics</a>, or mathematics for its own sake, without having any application in mind. There is no clear line separating pure and applied mathematics, and practical applications for what began as pure mathematics are often discovered.</p>

<h2 style="font-family:sans-serif; font-size:100%; font-weight:bold; border: none; margin: 0; padding:0; padding-bottom:.1em; color:#503b1e;"><span class="mw-headline" id="Selected_article">Selected article</span></h2>
</div>
<div style="border: 1px solid tan; border-top-width: 1px; vertical-align: top;background: white;opacity: 1; color: black; text-align: left; margin: 0 0 10px; padding: 1em;padding-top: .3em;">
<p>The <b>continuum hypothesis</b> is a <a href="/wiki/Hypothesis" title="Hypothesis">hypothesis</a>, advanced by <a href="/wiki/Georg_Cantor" title="Georg Cantor">Georg Cantor</a>, about the possible sizes of <a href="/wiki/Infinite_set" title="Infinite set">infinite sets</a>. Cantor introduced the concept of <a href="/wiki/Cardinal_number" title="Cardinal number">cardinality</a> to compare the sizes of infinite sets, and he showed that the set of <a href="/wiki/Integer" title="Integer">integers</a> is strictly smaller than the set of <a href="/wiki/Real_number" title="Real number">real numbers</a>. The continuum hypothesis states the following:</p>
<dl>
<dd>There is no set whose size is strictly between that of the integers and that of the real numbers.</dd>
</dl>
<p>Or mathematically speaking, noting that the <a href="/wiki/Cardinal_number" title="Cardinal number">cardinality</a> for the integers <img class="mwe-math-fallback-image-inline tex" alt="|\mathbb{Z}|" src="//upload.wikimedia.org/math/6/a/3/6a306e286a62fde11e660b45182c45f5.png" /> is <img class="mwe-math-fallback-image-inline tex" alt="\aleph_0" src="//upload.wikimedia.org/math/b/e/4/be4c703ed73456618ed283b892c6715a.png" /> ("<a href="/wiki/Aleph_number" title="Aleph number">aleph-null</a>") and the <a href="/wiki/Cardinality_of_the_real_numbers" title="Cardinality of the real numbers" class="mw-redirect">cardinality of the real numbers</a> <img class="mwe-math-fallback-image-inline tex" alt="|\mathbb{R}|" src="//upload.wikimedia.org/math/8/4/c/84c968a353ab0b3bd23442c996bf9ea9.png" /> is <img class="mwe-math-fallback-image-inline tex" alt="2^{\aleph_0}" src="//upload.wikimedia.org/math/a/b/5/ab543a3de71af10270830e31af27b8b8.png" />, the continuum hypothesis says</p>
<dl>
<dd><img class="mwe-math-fallback-image-inline tex" alt="\nexists \mathbb{A}: \aleph_0 &lt; |\mathbb{A}| &lt; 2^{\aleph_0}." src="//upload.wikimedia.org/math/2/3/2/2324d053fc0a5bc217909048e3546d72.png" /></dd>
</dl>
<p>This is equivalent to:</p>
<dl>
<dd><img class="mwe-math-fallback-image-inline tex" alt="2^{\aleph_0} = \aleph_1" src="//upload.wikimedia.org/math/9/2/4/92406cdb040720501d2908e2b8e7361e.png" /></dd>
</dl>
<p>The real numbers have also been called <a href="/wiki/Real_line" title="Real line"><i>the continuum</i></a>, hence the name.</p>

<p>A <b><a href="/wiki/Line_integral" title="Line integral">line integral</a></b> is an <a href="/wiki/Integral" title="Integral">integral</a> where the <a href="/wiki/Function_(mathematics)" title="Function (mathematics)">function</a> to be integrated, be it a <a href="/wiki/Scalar_field" title="Scalar field">scalar field</a> as here or a <a href="/wiki/Vector_field" title="Vector field">vector field</a>, is evaluated along a <a href="/wiki/Curve" title="Curve">curve</a>. The value of the line integral is the sum of values of the field at all points on the curve, weighted by some scalar function on the curve (commonly <a href="/wiki/Arc_length" title="Arc length">arc length</a> or, for a vector field, the <a href="/wiki/Dot_product" title="Dot product">scalar product</a> of the vector field with a <a href="/wiki/Differential_(infinitesimal)" title="Differential (infinitesimal)">differential</a> vector in the curve). A <a href="//commons.wikimedia.org/wiki/File:Line_integral_of_scalar_field.gif#Full_description_.28English.29" class="extiw" title="commons:File:Line integral of scalar field.gif">detailed explanation of the animation</a> is available. See also <a href="/wiki/File:Line_integral_of_vector_field.gif" title="File:Line integral of vector field.gif">a similar animation</a> illustrating a line integral of a vector field. Many formulas in elementary <a href="/wiki/Physics" title="Physics">physics</a> (for example, <i>W</i>&#160;=&#160;<i><b>F</b></i>&#160;·&#160;<i><b>s</b></i> to find the <a href="/wiki/Work_(physics)" title="Work (physics)">work</a> done by a constant force in moving an object through a displacement) have line integral versions that work for continuously varying quantities (for example, <i>W</i>&#160;=&#160;∫<sub>C</sub>&#160;<i><b>F</b></i>&#160;·&#160;<i>d<b>s</b></i> to find the work done in moving an object along a curve <i>C</i> within a gravitational or electric <a href="/wiki/Field_(physics)" title="Field (physics)">field</a>).</p>

<div style="border: 1px solid tan; border-top-width: 1px; vertical-align: top;background: white;opacity: 1; color: black; text-align: left; margin: 0 0 10px; padding: 1em;padding-top: .3em;">
<div class="floatright"><a href="/wiki/File:Nuvola_apps_filetypes.svg" class="image" title="Did you know..."><img alt="Did you know..." src="//upload.wikimedia.org/wikipedia/commons/thumb/4/44/Nuvola_apps_filetypes.svg/50px-Nuvola_apps_filetypes.svg.png" width="50" height="50" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/4/44/Nuvola_apps_filetypes.svg/75px-Nuvola_apps_filetypes.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/4/44/Nuvola_apps_filetypes.svg/100px-Nuvola_apps_filetypes.svg.png 2x" data-file-width="128" data-file-height="128" /></a></div>
<ul>
<li>...that a nonconvex <a href="/wiki/Polygon" title="Polygon">polygon</a> with three <a href="/wiki/Convex_set" title="Convex set">convex</a> vertices is called a <a href="/wiki/Pseudotriangle" title="Pseudotriangle">pseudotriangle</a>?</li>
<li>...that it is possible for a three dimensional figure to have a finite <a href="/wiki/Volume" title="Volume">volume</a> but infinite <a href="/wiki/Surface_area" title="Surface area">surface area</a>? An example of this is <a href="/wiki/Gabriel%27s_Horn" title="Gabriel's Horn">Gabriel's Horn</a>.</li>
<li>... that as the <a href="/wiki/Dimension" title="Dimension">dimension</a> of a <a href="/wiki/Hypersphere" title="Hypersphere">hypersphere</a> tends to infinity, its "<a href="/wiki/Volume" title="Volume">volume</a>" (content) tends to 0?</li>
<li>...that the primality of a number can be determined using only a single division using <a href="/wiki/Wilson%27s_Theorem" title="Wilson's Theorem" class="mw-redirect">Wilson's Theorem</a>?</li>
<li>...that the line separating the <a href="/wiki/Numerator" title="Numerator" class="mw-redirect">numerator</a> and <a href="/wiki/Denominator" title="Denominator" class="mw-redirect">denominator</a> of a <a href="/wiki/Fraction_(mathematics)" title="Fraction (mathematics)">fraction</a> is called a <b><a href="/wiki/Solidus_(punctuation)" title="Solidus (punctuation)" class="mw-redirect">solidus</a></b> if written as a diagonal line or a <b><a href="/wiki/Vinculum_(symbol)" title="Vinculum (symbol)">vinculum</a></b> if written as a horizontal line?</li>
<li>...that a monkey hitting keys at <a href="/wiki/Statistical_randomness" title="Statistical randomness">random</a> on a typewriter keyboard for an infinite amount of time will <a href="/wiki/Infinite_Monkey_Theorem" title="Infinite Monkey Theorem" class="mw-redirect">almost surely type</a> the complete works of William Shakespeare?</li>
</ul>
