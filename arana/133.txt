			<h1 id="firstHeading" class="firstHeading" lang="en">Type species</h1>
									<div id="bodyContent" class="mw-body-content">
									<div id="siteSub">From Wikipedia, the free encyclopedia</div>

<p>In <a href="/wiki/Zoological_nomenclature" title="Zoological nomenclature" class="mw-redirect">zoological nomenclature</a>, a <b>type species</b> (<i>Species typica</i>) is the <a href="/wiki/Species" title="Species">species</a> name with which the name of a <a href="/wiki/Genus" title="Genus">genus</a> or subgenus is considered to be permanently taxonomically associated, i.e., the species that contains the biological <a href="/wiki/Type_(biology)" title="Type (biology)">type</a> specimen(s).<sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span>[</span>1<span>]</span></a></sup> A similar concept is used for suprageneric groups called a <a href="/wiki/Type_genus" title="Type genus">type genus</a>. In <a href="/wiki/Botanical_nomenclature" title="Botanical nomenclature">botanical nomenclature</a>, these terms have no formal standing under the <a href="/wiki/International_Code_of_Nomenclature_for_algae,_fungi,_and_plants" title="International Code of Nomenclature for algae, fungi, and plants">code of nomenclature</a>, but are sometimes borrowed from zoological nomenclature. In botany, the type of a genus is a specimen (or, rarely, an illustration).<sup id="cite_ref-ICN_2-0" class="reference"><a href="#cite_note-ICN-2"><span>[</span>2<span>]</span></a></sup></p>
<p>Every named genus or subgenus in zoology, whether or not currently recognized as <a href="/wiki/Valid_name_(zoology)" title="Valid name (zoology)">valid</a>, is theoretically associated with a type species. In practice, however, there is a backlog of untypified names (i.e., names which identify certain animals as belonging to separate species when in fact they belong to the same one).</p>
<h2><span class="mw-headline" id="In_use">In use</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Type_species&amp;action=edit&amp;section=1" title="Edit section: In use">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote">See also: <a href="/wiki/Types_in_zoology" title="Types in zoology" class="mw-redirect">Types in zoology</a></div>
<p>A type species is both a concept and a practical system that is used in the <a href="/wiki/Biological_classification" title="Biological classification" class="mw-redirect">classification</a> and nomenclature (naming) of animals. The value of a "type species" lies in the fact that it makes clear what is meant by a particular genus name. This is an important concept whenever a taxon containing multiple species must be divided into more than one genus; the type species automatically assigns the name of the original taxon to one of the resulting new taxa, thus reducing the potential for confusion.</p>
<p>The term "type species" is regulated in <a href="/wiki/Zoological_nomenclature" title="Zoological nomenclature" class="mw-redirect">zoological nomenclature</a> by article 42.3 of the <i><a href="/wiki/International_Code_of_Zoological_Nomenclature" title="International Code of Zoological Nomenclature">International Code of Zoological Nomenclature</a></i>, which defines a type species as the <a href="/wiki/Name-bearing_type" title="Name-bearing type">name-bearing type</a> of the name of a genus or subgenus (a "<a href="/wiki/Rank_(zoology)" title="Rank (zoology)" class="mw-redirect">genus-group name</a>") is the "type species". In the Glossary, type species is defined as</p>
<dl>
<dd>"The nominal species that is the name-bearing type of a nominal genus or subgenus".<sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">[<i><a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed"><span title="This claim needs references to reliable sources. (August 2015)">citation needed</span></a></i>]</sup></dd>
</dl>
<p>The type species permanently attaches a <a href="/wiki/Genus_(biology)" title="Genus (biology)" class="mw-redirect">genus</a> to its formal name (its generic name) by providing just one <a href="/wiki/Species" title="Species">species</a> within that genus to which the genus is permanently linked (i.e. the genus must include that species if it is to bear the name). The species name in turn is fixed, in theory, to a type specimen.</p>
<p>For example, the type species for the <a href="/wiki/Land_snail" title="Land snail">land snail</a> genus <i><a href="/wiki/Monacha" title="Monacha">Monacha</a></i> is <i><a href="/wiki/Monacha_cartusiana" title="Monacha cartusiana">Monacha cartusiana</a></i>. That genus is currently placed within the family <a href="/wiki/Hygromiidae" title="Hygromiidae">Hygromiidae</a>. The type genus for that family is the genus <i><a href="/wiki/Hygromia" title="Hygromia">Hygromia</a></i>.</p>
<p>The concept of the type species in zoology was introduced by <a href="/wiki/Pierre_Andr%C3%A9_Latreille" title="Pierre André Latreille">Pierre André Latreille</a>.<sup id="cite_ref-Dupuis_3-0" class="reference"><a href="#cite_note-Dupuis-3"><span>[</span>3<span>]</span></a></sup></p>
